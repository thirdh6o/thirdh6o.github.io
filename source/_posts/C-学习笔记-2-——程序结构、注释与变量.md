## 掌握C#程序结构
下面是一个简单的C#程序结构示例：
```csharp
using System;    

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}
```
using System 使用系统命名 System 空间中的类。  
namespace 名称空间是用来组织代码的容器，它是类和其他名称空间的容器。  
class 类是数据和方法的容器，它为程序带来了功能。在C#中运行的每一行代码都必须在一个类中。在我们的示例中，我们将类命名为Program。 
> 如果您学习过Java 或其他面向对象的编程语言，您会注意到C#的类与Java的类有一些相似之处。  

Console 是System系统名称空间的一个类，它有一个WriteLine() 方法，用于输出/打印文本。  
## 上面的示例中你需要注意的一些要点
### 顶级语句
顶级语句（Top-level statements）可以把“控制台程序的样板代码”省掉。
还记得我们在第一篇中使用的HelloWorld程序吗？
```csharp
Console.WriteLine("Hello, World!");
```
在这个示例中，我们只写了一行代码，它会自动输出"Hello, World!"。  
这个特性从 C# 9 开始支持，.NET 6/7/8 的控制台模板默认就用它。  
在下一篇笔记中，我们会详细介绍顶级语句的使用。

### 隐式引入
如果您观察细心的话，您会注意到在上面的程序中，using System 这一行是灰色的，看起来并没有被使用。  
理论上，我们的代码调用了System.Console.WriteLine() 方法，应该需要 using System。  
这是因为现代 .NET 项目默认启用了「隐式 using」。从 .NET 6 / C# 10 开始，编译器会自动帮你加上一些常用的 using 语句，例如：
```csharp
using System;
using System.Collections.Generic;
using System.Linq;
using System.Threading.Tasks;
```
所以即使我们自己没有写 using System，编译器也会自动加上它，这就是它变成灰色的原因，编译器并没有使用这行语句。  
## WriteLine 或 Write
这是我们目前使用的两个输出方法，它们的区别在于是否换行。
```csharp
Console.WriteLine()   //方法用于输出/打印文本并换行。  
Console.Write()      //方法用于输出/打印文本，但不换行。  
```
您可以根据需要选择使用这两个方法。  
## 注释
注释是用来解释代码的，它不会被编译器执行。  
在C#中，有两种类型的注释：  
### 单行注释
单行注释以 // 开头，直到行尾结束。  
```csharp
// 这是一个单行注释
Console.WriteLine("Hello, World!");
```
### 多行注释
多行注释以 /* 开头，以 */ 结尾。  
```csharp
/*
这是一个多行注释
它可以跨越多行
*/
Console.WriteLine("Hello, World!");
```
## 变量
变量是用来存储数据的容器。在C#中，您需要先声明一个变量，然后才能使用它。  
### 变量声明
变量声明的一般形式为：
```csharp
数据类型 变量名;
```
例如：
```csharp
int age;
```
这声明了一个名为 age 的整数变量。  
### 变量赋值
变量赋值的一般形式为：
```text
变量名 = 值;
```
例如：
```csharp
age = 25;
```
这将把 25 赋值给 age 变量。  
### 变量使用
一旦变量被声明并赋值，您就可以在程序中使用它。  
例如：
```csharp
Console.WriteLine(age);
```
这将输出 age 变量的值，即 25。  
## 常量
常量是一种特殊的变量，它的值在程序运行过程中不能改变。  
在C#中，您需要先声明一个常量，然后才能使用它。  

>这就像您使用Word文档中的只读模式，常量的值一旦被赋值，就不能再改变。  

常量声明的一般形式为：
```csharp
const 数据类型 常量名 = 值;
```
例如：
```csharp
const int MaxAge = 120;
```
这声明了一个名为 MaxAge 的整数常量，它的值为 120。
